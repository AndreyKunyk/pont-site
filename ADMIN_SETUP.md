# Админ-панель KUNIK

Адрес панели после развёртывания:

`https://pont-site.onrender.com/admin`

## Обязательные переменные Render

- `ADMIN_PASSWORD` — длинный пароль администратора.
- `ADMIN_SESSION_SECRET` — случайная строка не короче 32 символов.

## Постоянное хранение наличия

Для production создайте Render Postgres и добавьте его Internal Database URL в переменную:

- `INVENTORY_DATABASE_URL`

Без этой переменной панель будет работать через SQLite, но на бесплатном Render изменения могут сбрасываться после перезапуска или нового deploy.

## Как пользоваться

1. Откройте `/admin`.
2. Введите пароль из `ADMIN_PASSWORD`.
3. Переключите товар в состояние «В наличии» или «Нет в наличии».
4. Сайт обновляет наличие при открытии и затем раз в минуту.
5. Недоступный товар нельзя добавить в корзину или оплатить: сервер повторно проверяет наличие при создании платежа.
